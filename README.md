# Proyecto Vivero

## Descripción del Proyecto
Sitio Web usando HTML y CSS
- index.html: Muestra 3 enlaces hacia las páginas productos.html; tiendas.html y registrarse.html. También muestra unas imagenes breves con las ofertas del mes.
- productos.html: Muestra los productos que vende el vivero, con una imagen y precio del producto.
- tiendas.html: Muestra en un mapa la ubicación de las diferentes tiendas en el país.
