# Proyecto Vivero

## Descripción del Proyecto

Sitio Web usando HTML y CSS

- index.html: Muestra 3 enlaces hacia las páginas productos.html; tiendas.html y registrarse.html. También  muestra unas imágenes breves con las ofertas del mes.

- productos.html: Muestra los productos que vende el vivero, con una imagen y precio del producto.

- tiendas.html: Muestra en un mapa la ubicación de las diferentes tiendas en el país.



| Tecnologias |Descripción  |
|--|--|
|HTML | Se uso una estructura html para la fomación de cada página, tomando en cuenta las buenas prácticas en el desarrollo de cada una de ellas. |
|CSS|  Se uso CSS para agregar estilo y color a cada una de las páginas creadas en el sitio web. Así como también se ajusto el tamaño de las imágenes para que quedaran de manera más adecuada posible.
